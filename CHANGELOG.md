## [1.0.0] - 2026-04-08

### Added
- ✨ Core skill for extracting book methodologies
- 📦 Automatic generation of independent skill files for each methodology
- 🎯 Support for slash command invocation
- 📚 10 example skills from "The Lean Startup" (《精益创业》)
  - mvp - 最小可行产品
  - build-measure-learn - 构建-测量-学习循环
  - validated-learning - 经证实的认知
  - innovation-accounting - 创新核算
  - pivot - 转型
  - five-whys - 五个为什么
  - hypotheses-validation - 价值假设与增长假设
  - actionable-metrics - 避免虚荣指标
  - continuous-deployment - 持续部署
  - lean-canvas - 精益画布
- 📖 Comprehensive documentation
- 📄 MIT License
- 🤝 Contributing guidelines

### Features
- Web search integration for book information extraction
- Structured methodology organization
- Complete skill template with scenarios, steps, and outputs
- Multi-source verification for accuracy
