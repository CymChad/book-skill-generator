## 如何使用

### 方法 1: 通过斜杠命令

直接输入斜杠命令,例如:
```
/find-community
/validate-idea
/mvp
```

### 方法 2: 通过自然语言

直接描述你的需求,系统会自动匹配对应的 skill,例如:
```
"我想找创业方向"
"帮我测试这个商业点子值不值得做"
"我准备做第一款产品,不知道该做哪些功能"
```

### 方法 3: 使用主 skill 提取其他书籍

输入类似这样的请求:
```
"从《精益创业》提取方法论,生成对应的 skill"
"把《原子习惯》的方法论变成 skill"
```
