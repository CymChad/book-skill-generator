## 📝 发布步骤

### 1. 更新 package.json

将 `package.json` 中的仓库地址改为你的实际 GitHub 地址：

```json
{
  "repository": {
    "type": "git",
    "url": "https://github.com/你的实际用户名/book-skill-generator.git"
  },
  "bugs": {
    "url": "https://github.com/你的实际用户名/book-skill-generator/issues"
  },
  "homepage": "https://github.com/你的实际用户名/book-skill-generator#readme"
}
```

### 2. 更新 README.md

将 README.md 中的所有 `你的用户名` 替换为你的实际 GitHub 用户名。

### 3. 初始化 Git 仓库

```bash
cd book-skill-generator
git init
git add .
git commit -m "feat: initial release of book-skill-generator"
```

### 4. 创建 GitHub 仓库

1. 登录 GitHub
2. 创建新仓库 `book-skill-generator`
3. 不要初始化 README、.gitignore 或 LICENSE（我们已经有了）

### 5. 推送到 GitHub

```bash
git remote add origin https://github.com/你的用户名/book-skill-generator.git
git branch -M main
git push -u origin main
```

### 6. 创建 Release

1. 在 GitHub 仓库页面点击 "Releases"
2. 点击 "Create a new release"
3. 填写信息：
   - Tag: `v1.0.0`
   - Title: `v1.0.0 - Initial Release`
   - Description: 参考 CHANGELOG.md

### 7. 添加主题标签

在 GitHub 仓库的 "About" 部分添加标签：
- skill
- openclaw
- dumate
- ai
- knowledge-extraction
- book-methodology
