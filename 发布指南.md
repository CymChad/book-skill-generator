# 发布指南

本指南将帮助你将 Book Skill Generator 发布到 GitHub，让用户可以通过 `npx skills add` 一键安装。

## 发布前准备

### 1. 完善 README.md

确保 README.md 包含以下内容：
- ✅ 项目简介和功能特点
- ✅ 安装方法（npx 和手动安装）
- ✅ 使用示例
- ✅ 生成的 skill 列表
- ✅ 适用场景
- ✅ 贡献指南
- ✅ 许可证信息

### 2. 准备示例文件

在 `examples/` 目录中放置生成的示例 skill，让用户了解输出效果。

### 3. 添加开源许可证

已添加 MIT 许可证，你可以根据需要修改。

## 发布步骤

### 步骤 1: 创建 GitHub 仓库

1. 登录 GitHub
2. 点击右上角的 "+" → "New repository"
3. 填写仓库信息：
   - Repository name: `book-skill-generator`
   - Description: `从书籍中提取核心方法论，生成可执行的 AI Skill`
   - 选择 Public
   - 勾选 "Add a README file"（如果还没有）
   - 选择 MIT License

### 步骤 2: 初始化 Git 仓库

```bash
cd book-skill-generator

# 初始化 Git 仓库
git init

# 添加所有文件
git add .

# 创建第一次提交
git commit -m "feat: initial release of book-skill-generator

- Add core skill for extracting book methodologies
- Add 10 example skills from 'The Minimalist Entrepreneur'
- Add comprehensive documentation
- Add MIT license"

# 添加远程仓库
git remote add origin https://github.com/你的用户名/book-skill-generator.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

### 步骤 3: 创建 Release

1. 在 GitHub 仓库页面，点击 "Releases" → "Create a new release"
2. 填写 Release 信息：
   - Tag version: `v1.0.0`
   - Release title: `v1.0.0 - Initial Release`
   - Description:
     ```markdown
     ## 🎉 首次发布

     ### 新功能
     - ✨ 从书籍中自动提取核心方法论
     - 📦 为每个方法论生成独立的可安装 skill
     - 🎯 支持通过斜杠命令直接调用
     - 📚 包含《小而美》10个方法论示例

     ### 安装方法
     ```bash
     npx skills add https://github.com/你的用户名/book-skill-generator
     ```

     ### 示例
     查看 `examples/` 目录中的《小而美》方法论 skill
     ```

### 步骤 4: 更新 package.json

修改 `package.json` 中的仓库地址：

```json
{
  "repository": {
    "type": "git",
    "url": "https://github.com/你的实际用户名/book-skill-generator.git"
  },
  "bugs": {
    "url": "https://github.com/你的实际用户名/book-skill-generator/issues"
  },
  "homepage": "https://github.com/你的实际用户名/book-skill-generator#readme"
}
```

## 用户安装方式

发布后，用户可以通过以下方式安装：

### 方式 1: npx 一键安装（推荐）

```bash
npx skills add https://github.com/你的用户名/book-skill-generator
```

或使用 Git 地址：

```bash
npx skills add git@github.com:你的用户名/book-skill-generator.git
```

### 方式 2: 手动安装

```bash
# 克隆仓库
git clone https://github.com/CymChad/book-skill-generator.git

# OpenClaw: 口语化安装
# 在 OpenClaw 中说：帮我安装下 https://github.com/CymChad/book-skill-generator

# Claude Code: 手动安装
mkdir -p ~/.claude/skills/
cp -r book-skill-generator ~/.claude/skills/
```

## 版本更新

### 更新流程

1. 修改代码
2. 更新 `package.json` 中的版本号
3. 提交代码：
   ```bash
   git add .
   git commit -m "feat: add new feature"
   git push
   ```
4. 创建新的 Release

### 版本号规范

遵循语义化版本规范：
- `MAJOR.MINOR.PATCH`
- MAJOR: 不兼容的 API 修改
- MINOR: 向下兼容的功能性新增
- PATCH: 向下兼容的问题修正

示例：
- `1.0.0` → `1.0.1`: 修复 bug
- `1.0.1` → `1.1.0`: 新增功能
- `1.1.0` → `2.0.0`: 重大更新
